$git init 
inicializa un nuevo repositorio Git en el directorio actual.
$git add 
agrega archivos al área de preparación para el commit.
$git commit 
registra los cambios en el repositorio local.
$git push 
envía los cambios locales al repositorio remoto.

Los datos son la clave para el éxito.
